석고계산기 모바일 최적화 - 변경내용 요약

=== 최종 목표 ===
그리기 도구 버튼들을 위쪽 메뉴처럼 가로로 한 줄에 배치하기

=== 주요 변경사항 ===

1. HTML 구조 변경
   - 그리기 도구를 캔버스 컨테이너 밖으로 이동
   - .main-menu 클래스 사용으로 위쪽 메뉴와 동일한 스타일 적용

2. 버튼 구성 변경
   - 기존 8개 버튼 → 6개 버튼으로 축소
   - 제거된 버튼: "삼각", "원", "정리"
   - 추가된 버튼: "이전" (되돌리기 기능)
   - 최종 구성: 선, 곡, 지우, 이전, 배경, 저장

3. CSS 스타일 최적화
   - .menu-item 프레임 크기 축소
     * min-width: 60px → 45px
     * padding: 8px 5px → 4px 2px
     * border-radius: 10px → 8px
   
   - 아이콘/텍스트 크기 축소
     * .menu-icon: 1.2rem → 1rem
     * .menu-text: 0.6rem → 0.55rem
     * margin-bottom: 3px → 1px

   - 텍스트 내용 단축
     * "곡선" → "곡"
     * "지우개" → "지우"

4. JavaScript 기능 추가
   - undoLastAction() 함수 추가
     * 마지막 그린 선 제거 기능
     * drawnLines.pop() + redrawAll()
     * 성공/실패 알림 메시지

5. showDrawingTools() 함수 수정
   - drawingTools와 canvasContainer 모두 표시
   - 캔버스 영역도 함께 나타나도록 수정

=== 현재 상태 ===
- PC에서는 6개 버튼이 가로로 잘 배치됨
- 모바일에서 테스트 필요 (GitHub 업로드 후)
- 모든 기능은 데스크탑 버전과 동일하게 유지
